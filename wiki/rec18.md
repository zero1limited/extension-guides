This report requires access to Google Analytics for the client account in question, or by using the zero1limited@gmail.com (password 01limiturD) account

1. Login to Google
2. [Load this Google Spreadsheet](https://docs.google.com/spreadsheets/d/1N4N3vG5t8WcYNCmgOvYShVokF65c5oCGAqHgOyfWAUs/edit#gid=1927412269)
3. In a separate window login to the Magento Admin panel of the sites (MDOQ UAT Instance) you are reporting on and carry out a complete inventory export from menu 'System > Import/Export > Export'
4. Import the exported data from the above, into [this Magexport document](https://docs.google.com/spreadsheets/d/1JfoavkB5y6ZpYe5IeST0pYgmT0Ktr8taNgcGCXgOjJE/edit#gid=660072343) ensuring you import using the 'Replace Current worksheet' option.
5. Select the 'Report Configuration' worksheet (yellow) and then select the required Website and then enter the number of days you are reporting on (see below for Reporting Periods).
6. Select Menu 'Add-ons > Google Analytics > Run Reports'
7. Wait for the Report Status Box to show that the reports have completed successfully and click 'OK'
8. Select Menu 'Magento PM Reporter > Process Data' and wait for the Running Script popup to disappear
9. Go to the 'VisibleSkus' worksheet and analyse the data.



## Reporting Periods

Having run a report on data over 30 days, you might find something insightful and interesting which requires further information and data. In these cases you can simple run the report again on a shorter or longer time period and see how the data changes.
