# Recaptcha setup for development
Recaptcha is something we should always have fully active in production however due to the nature, is something that we need to deactivate in development






## Instructions






Setup for Development Instances
If using MDOQ, you can bake the following settings into /mdoq/env.sh which will run on rollup.

```
# Disable Captcha
bin/magento config:set recaptcha_frontend/type_for/customer_login ''
bin/magento config:set recaptcha_frontend/type_for/customer_forgot_password ''
bin/magento config:set recaptcha_frontend/type_for/customer_create ''
bin/magento config:set recaptcha_frontend/type_for/newsletter ''
bin/magento config:set recaptcha_frontend/type_for/customer_edit ''
bin/magento config:set recaptcha_frontend/type_for/contact ''
bin/magento config:set recaptcha_frontend/type_for/product_review ''
bin/magento config:set recaptcha_frontend/type_for/sendfriend ''
bin/magento config:set recaptcha_frontend/type_for/place_order ''
bin/magento config:set recaptcha_frontend/type_for/paypal_payflowpro ''
```
